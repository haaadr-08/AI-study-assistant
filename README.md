# 📚 StudyMate

StudyMate is a web-based study assistant that helps students organize notes and generate study materials.

## ✨ Features

- Add study notes
- Generate summaries
- Generate quizzes
- Generate flashcards
- Save notes using Local Storage
- Search saved notes
- Sort notes (Newest / Oldest)
- Edit notes
- Delete individual notes
- Clear all notes
- Live character counter
- Live word counter
- Responsive design

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API
- Font Awesome
